# Vloggo Bot
A personal vlog coordinator that keeps track of everyone in the rotation and pings the next member when itâ€™s their time to vlog.

## Setup
* Clone the repo
* `cd` to repo
* create `.env` file with 
`DISCORD_TOKEN = Your Token`
`CHANNEL_ID = Your ID`
* `pip3 install -r requirements.txt`
* run `python main.py` to start the bot.
